PSP ISO COMPRESSOR

A tool created by TheMrCerebro
Copyright (C) 2022 TheMrCerebro


=============
Introduction:
=============

  With this tool you can compress a game in iso format used by a PSP
  with CFW. Simply indicate the name or path of the file to compress
  and voila!, the ISO will be compressed as much as possible.
